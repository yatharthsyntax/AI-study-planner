# AI Study Planner (Pro-level, Responsive)

This is a lightweight, client-side "AI-like" Study Planner project intended for a student portfolio.
Features:
- Generate a 14-day study plan using heuristics
- Spaced repetition review slots
- Export / Import JSON
- Mobile-friendly responsive layout
- Dark / Light theme toggle
- LocalStorage persistence

How to use:
1. Fill in subjects, hours per day, and optional exam dates.
2. Click "Generate Plan".
3. Export the plan to save or import later.
